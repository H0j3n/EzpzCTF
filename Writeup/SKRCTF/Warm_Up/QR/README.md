# Warm Up

Challenge Name : QR

# Solution

You can complete the challenge by scanning the QR code but let's use a tool for this.

Install this tool first
```bash
sudo apt-get install zbar-tools
```

Then run this command
```bash
zbarimg QR.png
```

We will get the flag!
